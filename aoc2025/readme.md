# Usage

- don't forget to install a virtualenv (will be based on pyenv current python version)

```python
  # Create a virtual environment
  python3 -m venv venv

  # Activate it (macOS/Linux)
  source venv/bin/activate

  # Deactivate
  deactivate
```
